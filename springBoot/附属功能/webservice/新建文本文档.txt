Apache Axis
    https://blog.csdn.net/zhouqixiang/article/details/1943096
     Apache Axis 是开源的 SOAP 服务器和客户端。
        SOAP 是网络间不同语言的程序进行交互的方法。
     SOAP 通常在 HTTP 上交换信息 : 客户端 POST 一个 SOAP 请求，然后接受 HTTP 响应， HTTP 的响应可能为成功代码、 SOAP 响应或者错误代码。
     SOAP 消息的格式是 XML ，在不同的 SOAP 系统之间使用这种消息格式来对信息进行结构化。
        消息在 SOAP Envelope 中包含一个或者多个 SOAP 元素， Header 或者 SOAP Body 。
     附：
        Axis 实现了 JAX-RPC API 。 JAX-RPC API 是 Java 服务编程的标准之一。
        AXIS支持的标准是：W3C SOAP,WSDL,SAAJ( SUN公司：SOAP with Attachments API for Java);JAX-RPC(SUN公司：Java API for XML-Based RPC)。
